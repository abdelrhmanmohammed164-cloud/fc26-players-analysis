# ⚽ FC26 Players Analysis Dashboard

An interactive Streamlit dashboard for analyzing FC26 player statistics.

## Features
- Top players by any statistic
- Player comparison
- Overall ratings
- Interactive charts
- Search and filters

## Technologies
- Python
- Pandas
- Matplotlib
- Streamlit
